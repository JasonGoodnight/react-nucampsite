Install | Use

Installation Instructions

  1. In Visual Studio Code, click the extension's icon in the left column toolbar.
  2. In the searchbar that appears, type live server.
  3. Select the Live Server Extension. It should be the top choice.
  4. In the page for live server appears, click green install button.

Usage Instructions

  • Visual Studio Code must be open be open to a folder to use live server.
  • In VS codes's explorer, right click (windows) or control+click (macOS) on an html file. From the context menu that appears, select open with live server.
